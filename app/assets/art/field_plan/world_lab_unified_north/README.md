# Unified North corridor cutouts

These transparent raster cutouts were generated on 2026-07-21 for the Brigade / Fields / North diorama lab. They form a single projected corridor rather than a separate North scene:

- `north-transition-ground-v1.png`: agricultural-to-snow ground plate
- `north-sparse-pines-v1.png`: paired snowbank and sparse-pine edge cards
- `north-forest-edge-v1.png`: irregular terminal forest edge
- `north-station-v1.png`: long station gate with a central route passage

The source prompts specified flat Soviet printmaking, restrained cream/ochre/olive/navy/red inks, paper grain, isolated compositions, and chroma-key backgrounds. The generated chroma masters were converted to transparent PNGs with the image-generation skill's chroma removal helper. The world renderer supplies the perspective, overlap, and parallax; these masters should remain unskewed.
