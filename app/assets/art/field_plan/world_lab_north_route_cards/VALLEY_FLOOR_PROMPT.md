# A09 supplemental valley-floor proof

Built-in image-generation edit using `world_lab_north_route_masters/rm36.png` as the
composition, palette, paper texture, and lithographic-style reference.

The first pass isolated a snow basin on a flat `#ff00ff` chroma-key background but was
too close to a full-width horizontal band. A targeted second edit preserved the surface
art while reshaping the matte into a centered forward valley: approximately 35% canvas
width near 38% image height, widening to approximately 70% near 94% image height.

Both prompts required restrained early-Soviet agricultural-poster lithography, the
locked 12:5 registration and vanishing point, natural irregular bleed, and no railway,
rails, sleepers, ballast, road, sky, forest, trees, buildings, fences, hut, text, or
signage. The final chroma source was converted to alpha with the installed imagegen
chroma-removal helper using a soft matte and despill.

Status: Flutter motion proof pending an editable Figma component and global-stack
instance. Do not treat this runtime file as the final art authority.
