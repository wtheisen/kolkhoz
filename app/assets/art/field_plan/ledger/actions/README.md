# Ledger Action Pictograms

These 512 x 512 transparent production assets support the create-game action strip:

- `save-favorite.png`
- `load-favorite.png`
- `go-to-lobby.png`

They were generated independently on a flat green background, processed with
`remove_chroma_key.py`, downsampled to 512 x 512, and validated for alpha.

Prompt direction: early-1930s agricultural-report or publishing-catalog spot
illustration; coal contour; aged cream, brick red, wheat ochre, and muted gray-blue;
ordinary people at human scale; no containing button, UI text, political emblem,
gradient, glow, or pixel art.

Selection, focus, progress, and disabled states remain code-driven rather than raster
assets.
